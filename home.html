Welcome to Image Resizer App

Resize your images with ease! Upload an image file or provide an image URL, and get a perfectly resized image delivered to your S3 bucket.

API Usage

Endpoint: POST /resize

Parameters:

• image - Image file upload (multipart/form-data)

• image_url - Image URL (JSON body)

• size - Output size (default: 3000)

• bg - Background color (white, transparent, or #RRGGBB)
